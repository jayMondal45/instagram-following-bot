Launch in 5 Steps

1

Clone Repository

Pull the latest version from GitHub to your local machine

Bash 
git clone https://github.com/jaymondal45/instagram-following-bot.git
cd instagram-following-bot
2

Install Dependencies

Ensure Python 3.9+ is installed, then install required packages:

Bash 
pip install selenium
3

Setup ChromeDriver

Download ChromeDriver matching your Chrome version and add it to your system PATH or project directory

4

Configure Parameters

Open main.py and customize the CONFIG dictionary:

Python 
CONFIG = {
    'username': 'your_username',
    'password': 'your_password',
    'target_account': 'target_profile_username',
    'max_follows': 2,
    'follow_delay': (25, 40)
}
username
Your Instagram account username
password
Your Instagram account password (stored locally)
target_account
Username of the account whose following list you want to target
max_follows
Maximum accounts to follow per session (recommended: 2-10)
follow_delay
Random delay range in seconds between each follow action
5

Launch Bot

Execute the automation and watch it work:

Bash 
python main.py

The bot will authenticate, navigate to the target profile, and begin following accounts with intelligent human-like behavior patterns.
